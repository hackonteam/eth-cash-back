## **‚öôÔ∏è How It Works (High Level)**

1. A dApp registers a cashback configuration:

   * Cashback percentage (e.g. 2%)

   * Cashback limit per user

   * Validity window

2. A user performs a transaction through ETH Cash Back.

3. The system:

   * Sponsors or simulates gas fees

   * Calculates eligible cashback

   * Transfers ETH cashback directly to the user

   * Enforces usage limits automatically

4. The user receives:

   * Successful transaction

   * ETH cashback

   * A friction-free experience
