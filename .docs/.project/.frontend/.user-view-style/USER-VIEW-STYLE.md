## Responsive Breakpoints

### Layout Changes

| Breakpoint | Layout Change |
|------------|---------------|
| < 640px | Full-width card, stacked elements |
| 640-1024px | Centered card, increased padding |
| > 1024px | Max-width container, comfortable spacing |

### Element Visibility

| Element | Mobile | Tablet | Desktop |
|---------|--------|--------|---------|
| Network text | Hidden | Visible | Visible |
| Full address | Hidden | Hidden | Visible |
| Truncated address | Visible | Visible | Hidden |

| Layer | Z-Index | Elements |
|-------|---------|----------|
| Base | 0 | Page content |
| Header | 10 | Fixed header |
| Dropdown | 20 | Wallet dropdown, tooltips |
| Modal backdrop | 30 | Modal overlay |
| Modal content | 40 | Modal itself |
| Toast | 50 | Notifications |

## Spacing Rhythm

Vertical spacing within card:
- Between sections: var(--space-6)
- Between related elements: var(--space-3)
- Between label and input: var(--space-2)
- Padding: var(--space-6)

## Alignment Guidelines

- Text: Left-aligned by default
- Numbers: Right-aligned in tables
- Buttons: Full-width in mobile, auto-width in desktop
- Cards: Center-aligned on page
- Modals: Center of viewport
